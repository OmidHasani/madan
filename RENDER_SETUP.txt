  راهنمای دپلوی روی Render.com - راهنمای ساده
═══════════════════════════════════════════════════════════════

📋 مراحل دپلوی:

1️⃣ آماده‌سازی:
   ✓ پروژه را به GitHub/GitLab push کنید
   ✓ فایل PDF را در root پروژه قرار دهید (یا از Persistent Disk استفاده کنید)

2️⃣ ایجاد سرویس در Render:
   - وارد render.com شوید
   - "New +" > "Web Service"
   - ریپازیتوری را متصل کنید

3️⃣ تنظیمات:
   
   Build Command:
   ────────────────────────────────────────────────────────────
   pip install -r requirements.txt && python initialize_db.py || echo "Database init skipped"
   ────────────────────────────────────────────────────────────
   
   Start Command:
   ────────────────────────────────────────────────────────────
   uvicorn api:app --host 0.0.0.0 --port $PORT
   ────────────────────────────────────────────────────────────

4️⃣ Environment Variables:
   - OPENAI_API_KEY = کلید OpenAI شما
   - TAVILY_API_KEY = کلید Tavily (اختیاری)

5️⃣ دپلوی:
   - روی "Create Web Service" کلیک کنید
   - منتظر بمانید (5-10 دقیقه)
   - URL سرویس شما آماده است!

═══════════════════════════════════════════════════════════════

💡 نکات مهم:

• Free Plan: بعد از 15 دقیقه sleep می‌شود
• Persistent Disk: برای نگهداری داده‌ها فعال کنید
• فایل PDF: باید در پروژه باشد یا از Persistent Disk استفاده کنید
